# 🚀 rust-IME 简易安装指南

欢迎使用 rust-IME！如果您是第一次在 Linux 上安装软件，请不要担心，按照下面的步骤操作即可。

---

## 第一步：解压文件
1. 找到您下载的 `rust-ime-linux-x64.tar.gz` 文件。
2. 在文件上点击鼠标右键，选择 **“提取到此处”** (Extract Here)。
3. 进入解压后生成的文件夹。

## 第二步：打开“黑窗口” (终端)
1. 在文件夹的空白处点击鼠标右键。
2. 选择 **“在终端中打开”** (Open in Terminal)。这时会弹出一个黑色的窗口。

## 第三步：执行安装命令
1. 在弹出的黑色窗口中，完整地输入以下命令（或者直接复制粘贴）：
   ```bash
   bash ./install.sh
   ```
2. 按下回车键 (Enter)。
3. **输入密码**：安装过程中可能会提示您输入密码（输入密码时，屏幕上不会显示任何字符，这是正常的，输完直接按回车即可）。

## 第四步：重启电脑（非常重要！）
安装完成后，黑色窗口会提示“安装完成”。
**请务必重启电脑或注销重新登录**，否则输入法可能无法识别您的键盘。

---

## ❓ 常见问题解决方法

### 0. 我的桌面环境支持吗？
- **支持最好**：KDE (Wayland)、COSMIC、Hyprland。
- **可能出问题**：GNOME 或 XFCE（界面可能无法正确悬浮）。
- **即使界面不出来**：打字功能通常也是能用的（只要权限配置正确）。

### 1. 提示 "Permission denied" (权限不足)
如果运行失败，请先输入以下命令再尝试安装：
```bash
chmod +x install.sh
```

### 2. 安装后打不开网页配置页面
安装完成后，在浏览器地址栏输入 `http://localhost:8765` 即可进入设置。如果打不开，请确保 `rust-ime` 程序正在运行。

### 3. 键盘无法打字了怎么办？
如果启动输入法后键盘没反应，通常是因为权限没生效。
- **解决方法**：请确保您已经重启过电脑。如果还是不行，请尝试在终端输入 `sudo ./install.sh` 重新安装一次。

### 4. 怎么卸载？
目前您可以直接删除解压的文件夹，并运行以下命令清理系统链接：
```bash
sudo rm /usr/local/bin/rust-ime
```
