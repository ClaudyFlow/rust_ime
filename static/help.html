使用帮助与文档

欢迎使用 rust-ime，这是一款极速、轻量且高度可定制的输入法引擎。本文档将指导您了解核心功能及英文辅码的使用。

基础操作指南

1. 模式切换

  • 中/英切换: 默认使用 Caps Lock 键。也可在设置中更改为 Ctrl + Space。
  • 方案切换: 默认使用 Ctrl + Alt + S 循环切换（如中文 -> 日语 -> 英语）。
  • 快捷方案跳转: 按下 ` (波浪号键) 进入快捷模式，然后按首字母跳转：
    • c -> Chinese (中文)
    • e -> English (英语)
    • j -> Japanese (日语)

2. 候选词选择

  • 数字键选词: 按 1 ~ 9 直接选择对应候选词。
  • 空格键:
    • 首选上屏: 默认上屏第一个候选词。
    • 长句模式: 在长句模式下，空格作为分词符，第二次按空格才上屏。
  • 翻页:
    • 默认使用 ↑ / ↓ (上下方向键) 翻页。
    • 也可在设置中开启 - / = 翻页。

英文辅码功能详解

rust-ime 引入了独特的“英文辅码”筛选机制，旨在解决拼音输入中重码过多的问题，让您无需翻页即可精准定位候选词。

功能原理

在输入拼音后，您可以直接输入大写字母作为辅码，用来筛选以此字母开头的英文单词（或候选词提示中包含该字母的词）。

格式： 拼音 + 大写首字母

使用示例

场景 1：输入 "号" (Hao -> Number/Call)

输入 hao，候选词可能有“好”、“号”、“豪”等。

想打“号” (对应英文 Call 或 Number)，输入 haoC。

结果：“号”会瞬间排到第一位。

场景 2：输入 "离" (Li -> Leave)

输入 li，重码极多（里、力、离...）。

想打“离” (对应英文 Leave)，输入 liL。

结果：“离”被筛选出来。

场景 3：精确筛选

如果辅码字母唯一，系统甚至会自动选中。

如果辅码对应多个词（如 haoC 可能对应 Call 和 Cry），则这些词都会被提前，按频率排序。

优点

  • 无需翻页: 对于常用字，通过辅码基本可以实现“一击必中”。
  • 语义关联: 辅码基于英文单词，符合直觉（如“爱”-> Love -> L，“看”-> See -> S），容易记忆。
  • 不干扰正常输入: 只有输入大写字母时才触发，正常拼音输入完全不受影响。

局限与缺憾

  • 需要英文基础: 用户需要知道汉字对应的常用英文单词（如知道“食”对应 Eat）。
  • Shift 键操作: 输入辅码需要按住 Shift 键（输入大写），这比单纯按小写字母多了一个动作。
  • 多义词: 有些汉字对应多个英文单词（如“行”可以 Walk, Go, OK, Line...），词库可能只收录了其中一部分。

其他高级功能

长句模式 (Double Space Mode)

这是 rust-ime 的默认推荐模式。输入一长串拼音，使用空格进行分词，屏幕上会实时显示转换结果。输入结束时，再次按空格确认上屏。这种方式非常适合整句输入，能大幅提高打字速度。

防呆模式

开启后，如果您输入的拼音组合在词库中不存在（例如 biang 或错打的 ingg），系统会阻止您继续输入无效字符，并可能发出提示音。这能有效减少无效击键。

长韵母映射 (双快)

通过 Tab + 首字母 快速输入长韵母。例如：

  • Tab + L -> iang (良 liang)
  • Tab + S -> ong (送 song)

您可以在配置中心的“长韵母映射”页面自定义这些组合。
