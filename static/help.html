rust-ime 终极指南

一款专为程序员与效率极客设计的拼音输入法

产品定位

rust-ime 是一款基于 Rust 编写的高性能拼音输入法。它的设计核心是“零干扰”与“极致效率”。

对于普通用户，它表现得像任何主流输入法一样：支持全拼、简拼、模糊音。但为了效率，我们做了两个果断的选择：

  • Tab 键即正义：摒弃了复杂的切换键，默认使用 Tab 键（单按）瞬间切换中英文。
  • 位置固定：候选框不跟随光标（减少视觉抖动），默认固定在屏幕底部（位置可完全自定义）。

核心：英文辅助码

这是 rust-ime 的灵魂功能。在拼音重码极多（如同音字、简拼）时，您无需翻页，只需输入一个大写字母即可精准定位。

它是如何工作的？

输入法会自动扫描候选词对应的英文翻译/释义。当您输入一个大写字母时，它会匹配释义中第一个大写单词的首字母。

场景 A：解决同音字重码

输入 li，候选词可能有“里、离、礼、理...”。

  • 输入 liC ⮕ 礼 (Ceremony) 瞬间排到第一。
  • 输入 liL ⮕ 理 (Logic) 瞬间排到第一。

场景 B：简拼精准定位

输入 sh，候选词可能有“书、水、数、树...”。

  • 输入 shB ⮕ 书 (Book) 瞬间出现。
  • 输入 shW ⮕ 水 (Water) 瞬间出现。
  • 输入 shT ⮕ 树 (Tree) 瞬间出现。

场景 C：长词组过滤

输入 ha，想输入“号”。

  • 输入 haN ⮕ 号 (Number) 排除掉“好、哈”。
技巧：该功能会自动跳过声调符号。例如翻译是 "ne Particle"，输入 P 即可精准匹配 "Particle"。

进阶：笔画输入法与辅助码 (SBSRF)

这是本引擎最核心的原创设计方案。它不仅可以作为拼音的辅助过滤码，更可以作为一套独立、高效的笔画输入法。其核心逻辑是：一键代表两个连续笔画。

1. 设计思想：双笔一键

传统的笔画输入法（1-5数字键）效率低下，因为每输入一个笔画就需要点击一次按键。SBSRF 方案通过 5x5 的字母矩阵，让您每一次击键都能输入两个笔画，大幅提升了输入速度。

2. 映射布局：5x5 视觉矩阵

我们将键盘划分为 5 个逻辑区（横、竖、撇、捺、折）。每个按键代表两笔：行代表第 1 笔，列代表第 2 笔。

3. 编码规则

  • 单字输入 (全码)：依次输入笔画对对应的字母。例如“的”的前两画为“撇、竖”（32=R），继续输入后续笔画即可。
  • 单字输入 (定长 4 键简码)：为了提高长字的录入速度，所有 8 笔以上（编码超过 4 键）的汉字支持简码输入：
    规则：取该字全码的 前 3 键 + 最后一个键。
    示例：康 全码为 dy...a，简码仅需输入 4 键。
  • 万能通配符 (Z 键)：当您不确定中间某处的笔画时，可以使用 z 键占位：
    规则：z 匹配任意一个编码字母（即 2 个笔画）。
    示例：输入 fzh 可匹配所有以 f 开头、h 结尾的三键编码（共 6 笔）。
  • 词组输入 (定长 4 键制)：借鉴五笔逻辑，无论词组多长，统一使用 4 个按键完成编码：
    • 2 字词：取第 1 字的前 2 键 + 第 2 字的前 2 键。
      示例：中国 ⮕ m(中1)+n(中2) + a(国1)+h(国2) = mnah
    • 3 字词：取第 1 字首键 + 第 2 字首键 + 第 3 字前 2 键。
      示例：计算机 ⮕ y(计1) + t(算1) + f(机1)+s(机2) = ytfs
    • 4 字及以上：取前 3 个字的首键 + 最后一个字的第一个键。
      示例：中华人民共和国 ⮕ m(中1) + r(华1) + r(人1) + a(国1) = mrra

4. 使用场景

实战演练

  • 人 (ren)：笔画为“撇、捺”(3、4)，对应坐标 (3,4) 即按键 W。输入 ren;w 瞬间定位。
  • 的 (de)：前两画“撇、竖”(3,2=R)，末两画“点、折”(4,5=P)。输入 de;rp 精准定位。
说明：如果某个字映射结果与直觉不符，可以在 Web 端“辅助码对照表”中查看每个字的精确 stroke_aux。

4. 专有名词原生隔离

在普通输入法中，输入 beijing 可能会混杂“背景”和“北京”。在 rust-ime 中：

  • 全小写：beijing 只会匹配普通词汇（如：背景）。
  • 首字母大写：Beijing 或 Be 将触发专有名词索引，精准匹配“北京”，完全隔离干扰。

这对于输入人名、地名、软件名非常有效，实现了索引层面的清爽。

5. 多方案混输

rust-ime 允许您在网页端自由组合活跃方案（如 中文+英文）。

  • 程序员友好：在写注释时，不需要切换模式，直接输入 git, cargo, sudo，它们会出现在候选词中。
  • 英文自动补全：支持英文前缀联想。输入 com 即可选择 complete, command 等词。

6. 长按与双击定制 New

rust-ime 提供了极致的按键行为定制能力，您可以利用双击或长按来实现极速输入。

1. 双击快捷输入 (Double-Tap)

快速点击同一个字母键两次，注入长韵母或常用短语。例如：

  • i, i ⮕ 注入 ing
  • u, u ⮕ 注入 sh

2. 长按行为定制 (Long Press)

按住字母键不放（约 400ms），触发预设操作。默认映射为该字母的大写形式。

配置提示：访问配置中心，点击可视化键盘上的字母，即可分别为“双击”和“长按”分配任何您想要的字符串、变音字符或音调符号。

7. 智能导航与平卷互换 New

1. HJKL 导航模式

在输入拼音的过程中，按下 ` (反引号) 即可进入导航模式。此时您可以像使用 Vim 一样操作候选词：

  • H / L：切换上一个/下一个候选词
  • J / K：向下/向上翻页

按任意其他字母键将自动退出导航并继续拼写。

2. 平卷舌一键互换 (/)

拼错平/卷舌音时无需删除重打。在输入 zan 时发现应该是 zhan？只需按一下 / 键，系统会自动完成互换。

3. 提示词上屏 (Shift + Space)

如果您想直接输入候选词旁边的英文提示或注音，只需按下 Shift + Space 即可将其直接上屏。

8. 智能长句模式

在“系统总览”中将上屏模式切换为“双空格上屏”即可进入长句模式。

在此模式下，输入法会尝试进行全句切分和联想。虽然目前仍处于完善阶段，但我们计划在未来引入基于语境的智能纠错，让长篇大论的输入更加流畅。

9. 视觉与样式定制

我们提供了一个极致自由的皮肤引擎，全部可通过网页控制中心调节：

  • 双模选择：传统横向窗 vs 现代侧边卡片窗。
  • 精准调色：点击颜色球，为背景、高亮、文字选择任何您喜欢的颜色（支持透明度）。
  • 字号调节：主词条与辅助码（Hint）的字号可以分开控制，满足不同的视力习惯。
